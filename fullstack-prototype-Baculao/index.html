Welcome to Full-Stack App

A static frontend prototype demonstrating role-based access, client-side routing, local persistence, and CRUD modules prior to full backend integration.

Key Simulated Features
  • Email registration + simulated verification flow
  • State simulation with authentication tokens
  • Role-based interface customization (Admin/User)
  • Full CRUD capabilities for Employees, Departments, and Requests
Note: All application data is cached in localStorage. Click refresh to reset state or clear storage to restore seeds.
